# Stress Detection and Management System

This project is a real-time stress detection system that uses machine learning to predict stress levels based on physiological data such as heart rate and body temperature.

The system integrates a trained ML model with a web application to provide instant predictions and user-friendly suggestions. It is designed to help users monitor their stress levels and take timely actions for better mental well-being.

## 🔍 Features
- Real-time stress prediction (Low, Medium, High)
- Machine Learning model (Random Forest + XGBoost + Logistic Regression)
- Interactive web interface
- Suggestion system based on stress level
- Ready for Arduino sensor integration

## 🛠 Tech Stack
- Python (Flask)
- Machine Learning (Scikit-learn, XGBoost)
- HTML, CSS, JavaScript
- Arduino (for real-time data)

## 🚀 How It Works
1. Collects heart rate and body temperature data
2. Sends data to backend (Flask API)
3. ML model predicts stress level
4. Displays results on web interface with suggestions

## 🌍 Future Scope
